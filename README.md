# 🚀 CQC Automation Platform

A Continuous Quality Control & Automation Engineering Project
simulating enterprise-grade QA, CI/CD, AI model validation, and performance testing.

---

## 🔥 Features

- Unit Testing (Pytest)
- API Testing (FastAPI TestClient)
- UI Automation (Playwright)
- Performance Testing (Locust)
- AI Model Validation
- Drift Detection Monitoring
- Dockerized Environment
- CI/CD Pipeline (GitHub Actions)
- Code Coverage Reporting
- Allure Reporting
- SonarQube Static Analysis

---

## 🏗️ Architecture

FastAPI → Automated Tests → CI Pipeline → Docker → ML Monitoring
